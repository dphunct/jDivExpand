Welcome!

jDivExpand is a simple little jQuery plugin I wrote to make it easy to show a portion of a potentially large piece of information as a small area and yet allow it to grow upon command.

jDivExpand is simple to use. You simply apply it to an HTML element. The CSS file defines minimum and maximum sizes as well as widths and other styling. It incorporates jQuery UI so it will fit in naturally with your jQuery UI page. It also optionally utilizes jScrollPane to provide scrolling of large content within both the minimized and maximized component.
